
# 🚗 Bluetooth Controlled RC Car using Arduino

A Bluetooth-controlled robotic car developed using Arduino Uno, HC-05 Bluetooth module, motor driver, and four DC motors. The car can be controlled wirelessly through an Android smartphone using a Bluetooth Controller application.

---

## Features

- Wireless Bluetooth Control
- Forward Movement
- Backward Movement
- Left Turn
- Right Turn
- Stop Function
- Easy to Control
- Beginner Friendly Arduino Project

---

## Components Used

- Arduino Uno
- HC-05 Bluetooth Module
- L298N/L293D Motor Driver
- 4 DC Motors
- Robot Chassis
- Wheels
- Battery Pack
- Jumper Wires
- Android Smartphone

---

## Software Used

- Arduino IDE
- Bluetooth Controller Android App

---

## Working Principle

The Android application sends Bluetooth commands to the HC-05 module.

The HC-05 communicates with the Arduino through serial communication.

Arduino processes each command and controls the motor driver to rotate the motors in the desired direction.

This allows the robot car to move wirelessly in real time.

---

## Bluetooth Commands

| Button | Command |
|---------|---------|
| Forward | F |
| Backward | B |
| Left | L |
| Right | R |
| Stop | S |
